# bogan.co.nz (static site)

A static, GitHub Pages-ready site for bogan.co.nz:
- Home, Shop, Collections, Field Guide, Manifesto, About, Contact
- Product grids powered by `assets/js/products.js`

## Run locally
Just open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a new GitHub repo (e.g. `bogan-site`)
2. Upload all files (or push via git)
3. In GitHub: Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` / folder: `/ (root)`
6. Save, then your site goes live.

## Add products
Edit `assets/js/products.js` → `PRODUCTS` array.

## Notes
GitHub Pages is static. For real ecommerce checkout, integrate Shopify, Snipcart, or Stripe later.
